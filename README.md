# Login and Registration Web Pages

## Project Description
This project contains simple **Login** and **Registration** web pages created using **HTML and CSS**.  
The pages allow users to register and then log in using their username and password.

This project demonstrates basic **frontend web development concepts**, including:
- HTML forms
- Input fields
- CSS styling
- Page navigation

---

## Features
- User Registration Form
- Login Page
- Simple and clean UI design
- Form validation using the **required** attribute
- Page redirection after form submission

---

## Project Files

```
Project Folder
│
├── Login.html

└── README.md
```

---

## Login Page

The **Login Page** allows users to enter their login credentials.

### Fields
- Username
- Password

After clicking **Submit**, the user is redirected to:

```
Earth Pulse.html
```

---

## Registration Page

The **Registration Page** allows new users to create an account.

### Fields
- Full Name
- Email
- Username
- Password

After clicking **Register**, the user is redirected to:

```
Login.html
```

so they can log in using their new account.

## Technologies Used

## How to Run the Project

1. Download or clone the project folder.
2. Make sure all files are in the same directory.
3. Open **Registration.html** in any web browser.
4. Fill the form and register.
5. After registration, log in through the **Login Page**.

This project is a **frontend-only project**.
